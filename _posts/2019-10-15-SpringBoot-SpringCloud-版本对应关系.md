刚开始研究SpringCloud和SpringBoot的时候，困惑点比较多，SpringBoot是从1.X.X到目前的2.X.X版本，但是SpringCloud基于SpringBoot，版本之间肯定是有匹配或者兼容方面问题，经过调验，现在总结如下：

| SpringBoot | SpringCloud      |
| ---------- | ---------------- |
| 1.2.X      | Angel            |
| 1.3.X      | Brixton          |
| 1.4.X      | Camden           |
| 1.5.X      | Dalston、Edgware |
| 2.0.X      | Finchley         |
| 2.1.X      | Greenwich        |
